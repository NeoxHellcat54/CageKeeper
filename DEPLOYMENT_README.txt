CageKeeper v0.2.5.1 deploy fix

IMPORTANT:
1. Do not upload this ZIP file itself to GitHub Pages.
2. Extract the ZIP.
3. Upload the extracted CONTENTS to the root of the Pages branch/folder, so index.html is at the root.
4. Commit all files, including the assets/cards folder and .nojekyll.
5. After GitHub Pages finishes deploying, open /card-asset-check.html on your site to verify the card image names.
6. If the old app still appears, open /reset-once.html once, then hard-refresh /index.html.

Card images are mapped in assets/cards/card-map.json.
